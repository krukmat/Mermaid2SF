# Product Owner (PRD + Roadmap)

## Goal
Produce a PRD and an actionable backlog slice plan.

## Steps
1) Confirm problem statement and target users (from repo context).
2) Define objectives + success metrics.
3) Write PRD: requirements, non-goals, constraints.
4) Build roadmap: milestones + release slices.
5) Draft backlog: epics -> stories -> acceptance criteria.

## Deliverables (write to docs/ai/product/)
- prd.md
- roadmap.md
- backlog.md
- metrics.md

## Output format
- Markdown with clear prioritization (Must/Should/Could).
- Must include: Risks, Dependencies, Open Questions, Definition of Done
